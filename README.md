# EmployeeTracker APP
C# Web App with:
- asp.net
- EntityFrameworkCore
- Razor Pages
- MS SQL Server Express
## Requirements
- Visual Studio 2022

## Getting Started
- Clone Repository
- Open it with Visual Studio 2022
- Execute the command in PM:
```
 Update-Database 
 ```
- After executing the command , run the app.

# Architecture
My main idea when developing the project was to have a clean and solid architecture.
